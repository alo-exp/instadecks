**lib/enum-lint.js — lintCjs(src, {filename}={}):**
```js
function lintCjs(src, { filename = 'render-deck.cjs' } = {}) {
  const re = /addShape\s*\(\s*['"](\w+)['"]/;
  const m = src.match(re);
  if (m) {
    const before = src.slice(0, m.index);
    const line = before.split('\n').length;
    const col = m.index - before.lastIndexOf('\n');
    throw new Error(
      `[enum-lint] ${filename}:${line}:${col} — addShape() called with string literal "${m[1]}". ` +
      `Use pres.shapes.${m[1].toUpperCase()} instead (CRT-15 / D-05).`
    );
  }
}
module.exports = { lintCjs };
```

**lib/title-check.js — validateTitle(title, opts={}):**
- Constants:
  - `BLOCKED = new Set(['overview','introduction','outline','agenda','summary','conclusion','q&a','thank you','background'])` (case-insensitive normalized).
  - `VERBS` — ~80 common verbs (grew, fell, hit, beat, missed, launched, shipped, will, can, must, should, drives, drove, drove, captures, increases, decreases, doubled, tripled, accelerates, breaks, builds, closes, expands, generates, leads, reaches, reduces, requires, reveals, scales, signals, surpasses, transforms, unlocks, validates, wins, ...).
- Returns `{ok:false, reason}` if `opts.action_title_override !== true` AND any of:
  - `title.trim().split(/\s+/).length < 3` → reason `'too short (need ≥3 words)'`.
  - normalized lowercase title is in `BLOCKED` → reason `'blocked phrase: "<word>"'`.
  - no token in lowercase title is in `VERBS` → reason `'no verb detected (action title required)'`.
- Else `{ok:true}`.

**lib/design-validator.js — validateDesignChoice({palette, typography, brief, designIdeas}):**
- Inputs: `palette = {primary, secondary, accent, ink, muted}` (all 6-char hex no `#`), `typography = {heading, body, mono?}`, `brief` (uses `tone` + `topic` for justification heuristic), `designIdeas` (pinned design-ideas.json — palettes + typography_pairs).
- Returns `{ok, violations: [{rule, message}]}`.
- Rules:
  - `R1-default-blue`: palette.primary in `{'0070C0','1F4E79','2E75B6'}` AND brief tone/topic does not include `corporate`/`blue`/`finance` → violation.
  - `R2-typography-pinned`: `{heading, body}` pair not present in `designIdeas.typography_pairs` → violation.
  - `R3-hex-shape`: any palette field that is not exactly 6 hex chars → violation.
- Pure function — no fs.
