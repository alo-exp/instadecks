Order of operations (LOCKED — RESEARCH Q3, P-09):
  1. validate top-level: doc is object; doc.schema_version matches /^1\./; doc.slides is array
  2. for each slide: validate slideNum (positive integer), title (string), findings (array)
  3. for each finding: validate ALL 10 required fields present + types + ranges + allow-sets
  4. throw on FIRST violation with structured message
  5. filter: keep only `f.genuine === true`
  6. collapse: build `{ sev: SEV_MAP[f.severity_reviewer], nx, ny, text }` annotation
  7. assemble per-slide `{ slideNum, title, annotations[] }`; skip slides with empty annotations after filter

Error message format (regex enforced by tests):
  /^slides\[\d+\]\.findings\[\d+\]\.[a-z_]+: .+ not in .+$/  (range/allow-set form)
  /^slides\[\d+\]\.findings\[\d+\]\.[a-z_]+: must be .+$/    (type form)
  /^slides\[\d+\]\.findings\[\d+\] missing required field "[a-z_]+"$/ (presence form)

Export shape: `module.exports = { adaptFindings, SEV_MAP };`

    3. Function `adaptFindings(doc)`:
       a. Throw if `doc` is not a non-null object.
       b. Throw if `doc.schema_version` is missing or does NOT match `/^1\./` — message `Unsupported findings schema version ${doc.schema_version}. /annotate supports 1.x.`.
       c. Throw if `!Array.isArray(doc.slides)` — message `findings.slides: must be array`.
       d. For each `slide` at index `sIdx`:
          - Throw if `!Number.isInteger(slide.slideNum) || slide.slideNum < 1` — `slides[${sIdx}].slideNum: must be positive integer`.
          - Throw if `typeof slide.title !== 'string'` — `slides[${sIdx}].title: must be string`.
          - Throw if `!Array.isArray(slide.findings)` — `slides[${sIdx}].findings: must be array`.
          - For each `f` at index `fIdx` (let `where = "slides[${sIdx}].findings[${fIdx}]"`):
            - For each key in REQUIRED_FINDING_FIELDS: if `!Object.prototype.hasOwnProperty.call(f, key)` throw `${where} missing required field "${key}"`.
            - If `typeof f.severity_reviewer !== 'string'` throw `${where}.severity_reviewer: must be string`.
            - If `!(f.severity_reviewer in SEV_MAP)` throw `${where}.severity_reviewer: ${f.severity_reviewer} not in {Critical,Major,Minor,Nitpick}`.
            - If `!VALID_CATEGORY.has(f.category)` throw `${where}.category: ${f.category} not in {defect,improvement,style}`.
            - If `typeof f.genuine !== 'boolean'` throw `${where}.genuine: must be boolean`.
            - If `typeof f.nx !== 'number' || f.nx < 0 || f.nx > 1` throw `${where}.nx: must be number in [0,1] (got ${f.nx})`.
            - If `typeof f.ny !== 'number' || f.ny < 0 || f.ny > 1` throw `${where}.ny: must be number in [0,1] (got ${f.ny})`.
            - If `typeof f.text !== 'string' || !f.text.length` throw `${where}.text: must be non-empty string`.
            - For `rationale`, `location`, `standard`, `fix`: if `typeof f[key] !== 'string'` throw `${where}.${key}: must be string`.
       e. After ALL validation passes (no early-return), do a SECOND traversal:
          - Build `samples = []`.
          - For each slide: build `annotations = slide.findings.filter(f => f.genuine === true).map(f => ({ sev: SEV_MAP[f.severity_reviewer], nx: f.nx, ny: f.ny, text: f.text }))`.
          - If `annotations.length > 0` push `{ slideNum: slide.slideNum, title: slide.title, annotations }` to `samples`.
       f. Return `samples`.
    4. Export: `module.exports = { adaptFindings, SEV_MAP };`

    **Critical:** Validation traversal completes BEFORE filter traversal (P-09). Do NOT filter inside the validation loop.
