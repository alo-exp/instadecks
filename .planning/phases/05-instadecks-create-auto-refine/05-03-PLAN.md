     ```
     # Cycle N (1-indexed)
     1. If checkInterrupt(runDir): finalize ledger with ended_via='interrupted', write rationale, exit.
     2. Decide review_mode for this cycle:
        - cycle == 1 → 'full'
        - cycle == 2 AND prior cycle had findings_genuine == 0 → 'full'  (D-07 confirmation)
        - else → 'diff-only'
     3. Author/edit render-deck.cjs (cycle 1: from cookbook + brief; cycle 2+: edit to fix the prior cycle's genuine && severity ∈ {Critical, Major} findings, EXCLUDING any id present in any prior cycle's skipped_finding_ids).
     4. Invoke runCreate({...}) → produces <runDir>/deck.pptx + <runDir>/cycle-${N}/render-deck.cjs.
     5. Invoke pptx-to-images.sh → <runDir>/cycle-${N}/slides/slide-NN.jpg.
     6. Compute slidesToReview:
        - if review_mode === 'full' → null
        - else → slidesChangedSinceLastCycle(runDir, cycle)
     7. Invoke runReview({deckPath, runId, outDir: cycleDir, mode: 'standalone', slidesToReview}) → cycle-${N}/findings.json.
     8. Triage: read each finding alongside its slide image + the brief; for each, set:
        - id = `${slideNum}-${sha1(text).slice(0,8)}`  (stable across cycles)
        - genuine = boolean
        - triage_rationale = "1-2 sentence justification"
        Write to cycle-${N}/findings.triaged.json. Only `genuine && severity ∈ {Critical, Major}` flow into the next cycle's fix list.
     9. Compute issue_set_hash = hashIssueSet(<unfixed-genuine-finding objects with slideNum + text>).
     10. Append ledger entry: appendLedger(runDir, {cycle, timestamp, findings_total, findings_genuine, findings_fixed: <count of items the agent will fix in cycle N+1>, findings_intentionally_skipped, issue_set_hash, skipped_finding_ids: [...non-genuine ids], fixed_finding_ids: [...genuine-major ids], slides_changed, review_mode, ended_via: null}).
     11. Read ledger; detectOscillation(ledger) → if true, set last entry's ended_via = 'oscillation' (re-append a closing entry — JSONL is append-only), surface the ledger to the user, exit.
     12. Convergence check:
        - genuine_genuine_critical_or_major == 0 AND cycle >= 2 → set ended_via='converged', exit loop.
        - cycle == 1 AND genuine == 0 → continue to cycle 2 (forced confirmation per D-07).
     13. Soft-cap check: if cycle >= 5 AND not converged → AskUserQuestion (verbatim text below); route per response.
     14. Increment cycle; loop to step 1.
     ```

     ```
     D. Stop and let me review the ledger.
     ```

     ```
     c. Convert deck.pptx → deck.pdf via soffice (per Phase 4 runCreate).

     e. If the final findings union is empty (clean convergence), SKIP annotated artifacts and surface to user: "Clean convergence — no annotation overlay generated." (Pitfall 7).
     f. Surface the 8-artifact bundle to the user.
     ```
